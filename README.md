HappySoul – Mental Wellness Web Application 💚
HappySoul is a full-stack mental wellness platform designed to help users track their emotional well-being while providing admins with powerful management tools.

🌟 Features

👤 User Features

User Registration & Login
Mood Tracking (Happy, Sad, Anxious, etc.)
Secure JWT Authentication
Edit Profile
Change Password
Personalized Dashboard
Voice Based AI assistant for consultant


🛠 Admin Features

Admin Login System
Admin Dashboard
Mood Tracker Panel (View All User Mood Entries)
Manage Contact Queries
Admin Profile Management
Edit Profile
Change Password
Secure Role-Based Access Control


🔐 Authentication & Security

JWT-based authentication
Role-based navigation (Admin/User)
Protected routes
Secure token handling


🧩 Tech Stack

Frontend
React.js
React Router DOM
Bootstrap + Custom CSS
Axios
Backend
Node.js
Express.js
MongoDB (Mongoose)
JWT Authentication


📊 Mood Tracker Module

Users submit moods from dashboard
Mood data stored in MongoDB
Admin can view:
Mood
Emoji
User ID
Timestamp
Clean tabular UI for admins
